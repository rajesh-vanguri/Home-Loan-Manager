# Home Loan Manager

A beautiful, offline-first home loan tracker built as a single HTML file. No installation, no server, no dependencies except Chart.js.

## Features
Full amortization schedule with monthly & yearly views
Rate change tracking (floating rate loans)
Prepayment impact calculator
What If simulator: see how prepayments affect your payoff date
Best time to prepay insights
Dark mode
Auto-saves to localStorage: data persists across sessions
Export/Import data as JSON
Save charts as PNG images
Mobile-friendly with bottom navigation

## How to use
1. Download `index.html`
2. Open in any browser — no setup needed
3. Or visit the live version: [Home Loan Manager](https://rajesh.vanguri.github.io/home-loan-manager/home_loan_manager.html)

## Built with
Vanilla HTML, CSS, JavaScript
[Chart.js](https://www.chartjs.org/) for charts
localStorage for persistence
